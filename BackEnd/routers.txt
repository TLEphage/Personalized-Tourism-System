通用模块（待定）


用户模块
GET     /                                               -> 服务状态检查
POST    /users/register                                 -> 用户注册
POST    /users/login                                    -> 用户登录
GET     /users/{username}/details                       -> 查询用户信息
PUT     /users/{username}                               -> 更新用户信息

日记模块
POST    /diaries                                            -> 添加日记
GET     /diaries/{username}?sort_key=...&sort_order=...     -> 获取用户日记
PUT     /diaries                                            -> 更新日记

景点模块
GET     /spots/{name}?tag=...&sort_key=...&sort_order=...   -> 获取景点

地图模块
GET     /map/get_graph                                  -> 获取地图完整信息
POST    /map/add_node                                   -> 添加地图节点
POST    /map/add_edge                                   -> 添加地图边 
POST    /map/path_plan                                  -> 路径规划
POST    /map/search_places                              -> 场所查询
POST    /map/path_plan/indoor                           -> 室内导航

美食模块
POST     /foods/search                                  -> 美食搜索
