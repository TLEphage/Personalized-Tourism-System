## **本项目实现的功能：**
- 旅游推荐
- 旅游路线规划
- 场所查询
- 旅游日记管理
- 旅游日记交流
- 室内导航
- 美食搜索
- 内容推荐

**内容包含：**
- 景区和校园数量至少200个，景区和校园内部可以一致
- 景区和校园内建筑物（景点、教学楼、办公楼、宿舍楼）数不少于20个
- 其它服务设施不少于10种（商店、饭店、洗手间、图书馆、食堂、超市、咖啡馆等），数量不少于50个
- 建立景区和校园内部道路图，包括各种建筑物、服务设施等信息，边数不能少于200条
- 系统用户数不少于10人
